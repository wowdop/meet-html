padding-left and padding-right in CSS

padding-left

The padding-left property sets the left padding (space) of an element..

Note: Negative values are not allowed.

Example using padding-left:

Look at my left as there is some padding

Code example:

	<style>
	.left {
		padding-left: 15px;
		border: 2px ridge tomato;
	}
  </style>

	<div class="left">Look at my left as there is some padding</div>

padding-right

The padding-rightproperty sets the right padding (space) of an element.

Note: Negative values are not allowed.

Example using padding-right:

Look at my right as there is some padding

Code example:

  <style>
	.right {
		padding-right: 30px;
		width: 100px;
		background-color: yellow;
	}
  </style>

	<div class="right">Look at my right as there is some padding</div>
