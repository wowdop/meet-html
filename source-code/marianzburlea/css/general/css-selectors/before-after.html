::before and ::after in CSS

::before

The ::before selector inserts something before the content of each selected element(s).

Use the content property to specify the content to insert.

Use the ::after selector to insert something after the content.

Note: If you leave the value of content to be: " " then you need to add display property with the value of block like in my example, to create bullet points.

Example using :active

I will add a circle using the ::before property.

Code example:

  <style>
	.before::before {
		content: ' ';
		display: block;
		position: absolute;
		width: 10px;
		height: 10px;
		border-radius: 50%;
		background-color: green;
		left: 10px;
	}

	.before {
		padding-left: 16px;
	}
  </style>

  <div class="before">
    I will add a circle using the <b>::before</b> property.
  </div>

::after

The ::after selector inserts something after the content of each selected element(s).

Use the content property to specify the content to insert.

Use the ::before selector to insert something before the content.

Example using ::active

This is a reminder, so after me it will be some colourful text to express that.

Code example:

  <style>
    .after::after {
      content: 'REMEMBER ME';
      background-color: yellow;
      color: red;
    }
  </style>

  

    This is a reminder, so after me it will be some colourful text to express
    that.
