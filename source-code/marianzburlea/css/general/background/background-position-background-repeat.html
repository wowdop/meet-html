background-position and background-repeat in CSS

background-position

The background-position property sets the starting position of a background image.

Tip: By default, a background-image is placed at the top-left corner of an element, and repeated both vertically and horizontally.

Example using background-position:

I added a background picture that is placed in the right corner in the body tag style.

I also made use of background-repeat css selector.

Code example:

  <style>
		body {
			background-image: url('https://picsum.photos/id/1/200/300');
			background-attachment: fixed;
			width: 200px;
			height: 200px;
		}
  </style>

background-repeat

The background-repeat property sets if/how a background image will be repeated.

By default, a background-image is repeated both vertically and horizontally.

Tip: The background image is placed according to the background-position property. If no background-position is specified, the image is always placed at the element's top left corner.

Example using background-repeat:

I added a background picture that is placed in the right corner in the body tag style.

I made use of background-repeat: norepeat; in the style so that the image does not repeat across the whole space that is has available.

Code example:

  <style>
		.image {
			background-image: url('https://picsum.photos/200/300?grayscale');
			background-repeat: no-repeat;
			height: 300px;
			}
  </style>
