# CSS border

The CSS border properties allow you to specify the style, width, and color of an element's border.

The **border** CSS property is a shorthand property for setting:

- border-width,
- border-style,
- border-color.

Like all shorthand properties, a missing value for one of the longhand properties is set to the corresponding initial value.

<b>Note:</b>

- The border-image can't be set using this shorthand, but can be reset to its initial value, that is none.
- this property only accepts one value for each property, leading to the same border for all four edges.

## Example list

### Examples

```html:
  <style>
    img {
      border: 4px groove forestgreen;
    }

    h1 {
      border: 4px double steelblue;
    }
  </style>
```

# CSS border-top property, CSS border-right property, CSS border-bottom property & CSS border-left property

In CSS, there are also properties for specifying each of the borders:

- top,
- right,
- bottom,
- left.

The **border-top** CSS property is a shorthand that sets the values of:

- border-top-color,
- border-top-style,
- border-top-width.
  These properties describe the top border of elements.

The **border-right** CSS property is a shorthand that sets the values of:

- border-right-color,
- border-right-style,
- border-right-width.
  These properties describe the right border of elements.

The **border-bottom** CSS property is a shorthand that sets the values of:

- border-bottom-color,
- border-bottom-style,
- border-bottom-width.
  These properties describe the bottom border of elements.

The **border-left** CSS property is a shorthand that sets the values of:

- border-left-color,
- border-left-style,
- border-left-width.
  These properties describe the left border of elements.

## Example list

### Examples

```html:
  <style>
    .border-part-one {
        border-top: 6px solid goldenrod;
        border-right: 4px dotted magenta;
        border-bottom: 12px double maroon;
        border-left: 8px dashed lightseagreen;
      }
  </style>
```

# CSS border-radius

**border-radius** property is a shorthand to set the four properties:

- border-top-right-radius,
- border-top-left-radius,
- border-bottom-right-radius
- border-bottom-left-radius.

The **border-radius** CSS property defines the radius of the element's corners.

<b>Note:</b>

- This property allows you to add rounded corners to elements!
- The radius applies to the whole background, even if the element has no border.

This property can have from one to four values:

- Four values - border-radius, they apply in this order:

  - first value applies to top-left corner,
  - second value applies to top-right corner,
  - third value applies to bottom-right corner,
  - fourth value applies to bottom-left corner.

- Three values - border-radius, they apply in this order:

  - first value applies to top-left corner,
  - second value applies to top-right and bottom-left corners,
  - third value applies to bottom-right corner.

- Two values - border-radius, they apply in this order:

  - first value applies to top-left and bottom-right corners,
  - the second value applies to top-right and bottom-left corners.

- One value - border-radius: the value applies to all four corners, which are rounded equally.

# border-top-right-radius, border-top-left-radius, border-bottom-right-radius & border-bottom-left-radius Properties

The **border-top-right-radius** CSS property defines the radius of the top-right corner.

The **border-top-left-radius** CSS property defines the radius of the top-left corner.

The **border-bottom-right-radius** CSS property defines the radius of the bottom-right corner.

The **border-bottom-left-radius** CSS property defines the radius of the bottom-left corner.

## Example list

### Examples

```html:
  <style>
    .border-body {
        background-color: aquamarine;
        border-radius: 10%;
      }

      .p-one {
        border-top-left-radius: 16px;
        border: 4px solid blue;
      }


      .p-two {
        border-top-right-radius: 16px;
        border: 4px solid purple;
      }

      .p-three {
        border-bottom-left-radius: 16px;
        border: 4px solid orange;
      }

      .p-four {
        border-bottom-right-radius: 16px;
        border: 4px solid red;
      }
  </style>
```

# CSS border-color

**border-color** property is a shorthand to set the four properties:

- border-top-color
- border-right-color
- border-bottom-color
- border-left-color.

The **border-color** CSS property sets the color of an element's four borders. This property can have from one to four values:

- Four values, they apply in this order:

  - first value applies to top border,
  - second value applies to right border,
  - third value applies to bottom border,
  - fourth value applies to left border.

- Three values, they apply in this order:

  - first value applies to top border,
  - second value applies to right and left borders,
  - third value applies to bottom border.

- Two values, they apply in this order:

  - first value applies to top and bottom borders,
  - the second value applies to right and left borders.

- One value: the value applies to all four borders.

# border-top-color, border-right-color, border-bottom-color & border-left-color Properties

The **border-top-color** CSS property sets the color of an element's top border.

The **border-right-color** CSS property sets the color of an element's right border.

The **border-bottom-color** CSS property sets the color of an element's bottom border.

The **border-left-color** CSS property sets the color of an element's left border.

# CSS border-top-width, border-right-width, border-bottom-width & border-left-width

The **border-top-width** CSS property sets the width of an element's top border.

The **border-right-width** CSS property sets the width of an element's right border.

The **border-bottom-width** CSS property sets the width of an element's bottom border.

The **border-left-width** CSS property sets the width of an element's left border.

<b>Note: </b>

- Always declare the border-style property before the border-width property.
- An element must have borders before you can change the width.

## Example list

### Examples

```html:
  <style>
    .border-container {
      border-top-color: lightblue;
      border-right-color: red ;
      border-bottom-color: purple ;
      border-left-color: green;
      border-style: double;
      border-width: 4px;
    }

    .border-part-one {
      border-style: solid;
      border-color: blue;
      border-width: 4px 20px 16px 8px;
    }

    .border-part-two {
      border-style: solid;
      border-color: red yellow blue green;
      border-top-width: 4px;
      border-right-width: 20px;
      border-bottom-width: 16px;
      border-left-width: 8px;
    }
  </style>
```

# CSS border-image

**border-image** property allows you to specify an image to be used as the border around an element.

The border-image property is a shorthand property for:

- border-image-source: the path to the image to be used as a border.
- border-image-slice: how to slice the border image.
- border-image-width: the width of the border image.
- border-image-outset: the amount by which the border image area extends beyond the border box.
- border-image-repeat: Whether the border image should be repeated, rounded or stretched.

<b>Note:</b>

- The border-image is used instead of the border styles given by the border-style properties.
- It is important to note that if the computed value of border-image-source, which can be set either by border-image-source or the shorthand border-image, is none, or if the image cannot be displayed, the border styles will be used.
