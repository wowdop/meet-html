Display in CSS

The display property specifies the display behavior (the type of rendering box) of an element. In HTML, the default display property value is taken from the HTML specifications or from the browser/user default style sheet. The default value in XML is inline, including SVG elements.

Example using flex

Take a look at this example

There are two paragraphs here displayed in a column, with a background color.

I made use of display: flex and another CSS property called flex-direction that has only four values: row to place all the elements from inside in a row, column, that places all the elements in a column. row-reverse places the elements starting from the last and displays them in a row and column-reverse also starts from the last element and displays them in a column

Note: If you do not specify the direction, the default value is row

Code example:

	<style>
		.flex {
			display: flex;
			flex-direction: column;
			background-color: beige;
		}
	</style>

	<div class="flex">
		<p>Take a look at this example</p>
		<p>
			There are two paragraphs here displayed in a column, with a background
			color.
		</p>
	</div>

Example using grid

An HTML element becomes a grid container when its display property is set to grid or inline-grid.

All direct children of the grid container automatically become grid items.

The vertical lines of grid items are called columns.

The horizontal lines of grid items are called rows.

The spaces between each column/row are called gaps.

You can adjust the gap size by using one of the following properties:

  • column-gap
  • row-gap
  • gap

Code example:

This is a paragraph.

This is a different paragraph.

	<style>
	.grid {
		display: grid;
		grid-template-columns: repeat(2, 1fr);
		border: 2px dotted blue;
	}
	</style>

	<div class="grid">
		<p>This is a paragraph.</p>
		<p>This is a different paragraph.</p>
	</div>

justify-content and align-items

The justify-content property aligns the flexible container's items when the items do not use all available space on the main-axis (horizontally).

Tip: Use the align-items property to align the items vertically.

Value Description
flex-start Default value. Items are positioned at the beginning of the container
flex-end Items are positioned at the end of the container
center Items are positioned in the center of the container
space-between Items will have space between them
space-around Items will have space before, between, and after them
space-evenly Items will have equal space around them

The align-items property specifies the default alignment for items inside the flexible container.

Tip: Use the align-self property of each item to override the align-items property.

Value Description
stretch Default. Items are stretched to fit the container
center Items are positioned at the center of the container
flex-start Items are positioned at the beginning of the container
flex-end Items are positioned at the end of the container
baseline Items are positioned at the baseline of the container

Example using both

This is one paragraphs inside the container.

This is another paragraph and I want both of them to be in the middle of the element horizontally and vertically

Code example:

	<style>
	.container {
		display: flex;
		flex-direction: column;
		justify-content: center;
		align-items: center;
		border: 3px groove darkcyan;
		width: 50%;
		padding: 0 40px;
	}

	.color {
		background-color: red;
		color: white;
	}
	</style>

	<div class="container">
		<p class="color">This is one paragraphs inside the container.</p>
		<p class="color">
			This is another paragraph and I want both of them to be in the middle of the
			element horizontally and vertically
		</p>
	</div>
