Make use of style HTML tag

Add colours, sizes, text decorations etc.

The p tag will have some weird border and different colours.

Every tag can be selected and styled inside the style tag. Just write the tag and then open the curly brackets where you can start writing your CSS.

Example of code: 

    <style>
        h1 {
          color: yellowgreen;
          text-align: center;
        }
        h2 {
          font-size: 1rem;
          text-decoration: underline overline;
        }

        p {
          border: 2px dotted seagreen;
          background-color: wheat;
          color: brown;
        }
      </style>
