Multiple style tags

You can have multiple style tags but the computer reads starting from line 1 and going down to whatever number of lines you have.
Therefore, even if you have more styles for the same elements, the last one's values that are being read will win and your elements will get that style.
For example:

This is a sub-heading

This is a paragraph.

And the code looks like this:
    <style>
        h2 {color:red;}
        p {color:blue;}
    </style>
    <style>
        h2 {color:green;}
        p {color:pink;}
    </style>
