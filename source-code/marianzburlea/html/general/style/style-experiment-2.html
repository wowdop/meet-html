Classes in the style tag
div HTML tag can have the attribute class and you can give it any value that makes sense for you.

For example

I have given the class value "anna-background" for the div that contains the following text as it is a description about Anna.

Anna was born in Newcastle, UK in 1899. She had two elder sisters and she spent most of her childhood taking care of their farm.

Then I used the class anna-background in the style tag to give it a little life as following: 

    <style>
    .anna-background {
        border-top-left-radius: 1rem;
        border-bottom-right-radius: 1rem;
        background-color: darkseagreen;
        padding: 1rem;
        border: 1px dotted darkolivegreen;
        }
     </style>

Note: Always add a dot '.' so that you can select the class desired. ONLY actual HTML tags can be written without the dot
