menu tag simple experiments

Simple menu

In this example, menu is used to display a navigation through different pages on the left side of some content

The example below represent a very simple menu with two options, one to refresh the page one is for sending an email.

  • Reload
  • Email

    • I created a table in this example to display a menu on the left side of the content. It is very simple to reproduce


    Code example:

      <table border="2" cellspacing="10">
    <tr>
      <td>
        <menu>
          <li>
            <a href="#"&>Refresh</a>
          </li>

          <li>
            <a href="https://mail.google.com/mail/u/0/#inbox"&>Email</a>
          </li>
        </menu>
      </td>

      <td>
        <p>
        I created a table in this example to display a menu on the left side of
        the content. It is very simple to reproduce
        </p>
      </td>
    </tr>
  </table>

    Menu inside menu

    You can also add a menu tag inside a menu tag in order to create a submenu.

  • Profile
  • Settings
  • Recover
  • Pages
  • Home
  • News
  • Forum

    • I created a table in this example to display a menu on the left side of the content.

    It contains a menu inside a menu to create an appealing look and to differentiate the main menu labels and the submenu labels


    Code example:

      <table border="2" cellspacing="5" cellpadding="30">
    <tr>
      <td>
        <menu style="list-style: none">
          <li>
            <a href="#">Profile</a>
          </li>

          <menu style="list-style: none">
            <li>
              <a href="#">Settings</a>
            </li>

            <li>
              <a href="#">Recover</a>
            </li>
          </menu>

          <li>
            <a href="#">Pages</a>
          </li>

          <menu style="list-style: none">
            <li>
              <a href="#">Home</a>
            </li>

            <li>
              <a href="#">News</a>
            </li>

            <li>
              <a href="#">Forum</a>
            </li>
          </menu>
        </menu>
      </td>

      <td>
        <p>
          I created a table in this example to display a menu on the left side of
          the content.
        </p>

        <p>
          It contains a menu inside a menu to create an appealing look and to
          differentiate the main menu labels and the submenu labels
        </p>
      </td>
    </tr>
  </table>
