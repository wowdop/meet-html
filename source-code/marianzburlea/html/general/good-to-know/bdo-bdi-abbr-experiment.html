Code examples:

bdo examples:

  <p>
    <bdo dir="rtl">This text will go left to right. </bdo>
  </p>

  <p>
    <bdo dir="ltr">This text will go right to left. </bdo>
  </p>

bdi examples:

  <ul>
    <li>User <bdi>hrefs</bdi>: 13 projects</li>
    <li>User <bdi>jdoe</bdi>: 15 projects</li>
    <li>User <bdi>إيان</bdi>: 9 projects</li>
  </ul>

  <ul>
    <li><bdi>اَلأَعْشَى</bdi> - 1st place</li>
    <li><bdi>Jerry Cruncher</bdi> - 2nd place</li>
  </ul>

abbr examples:

  <p>
    Ashok's joke made me
    <abbr title="Laugh Out Loud">LOL</abbr>
    big time.
  </p>

  <p>
    <abbr title="HyperText Markup Language">HTML</abbr>
    is a markup language used to create the semantics and structure of a web page.
  </p>
