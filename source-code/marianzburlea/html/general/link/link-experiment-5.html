Use the hreflang attribute

In this example, we have used the hreflang attribute whose value is set to ”en-us” which will specify the language of the linked document. And we are linking an external CSS file that contains a color property for the h1 tag which is set to green.

Usage

It allows you to show search engines what the relationship is between web pages in alternate languages. It's useful when you've created content that's specific to a local audience.

Code example:

  <link
    rel="stylesheet"
    type="text/css"
    href="style-experiment-5.css"
    hreflang="en-us"
  />
