Change the font using link tag

Look at this text, how pretty it looks.

I made use of the link tag and a Google free font.

Code example:

    <head>
      <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Sofia" />

      <style>
        body {
          font-family: 'Sofia', sans-serif;
        }
      </style>
    </head>
