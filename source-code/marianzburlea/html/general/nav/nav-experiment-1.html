A very simple navigation menu

Using a tags inside the nav tag

  • Home
  • Html page
  • Profile
Code example:
  <nav>
    <ul>
      <li>
        <a href="/">Home</a>
      </li>
      <li>
        <a href="/marianzburlea/html">Html page</a>
      </li>
      <li>
        <a href="/marianzburlea">Profile</a>
      </li>
    </ul>
  </nav>
