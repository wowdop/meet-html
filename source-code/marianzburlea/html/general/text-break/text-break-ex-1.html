This is the most simple example of the use of br and hr tags

The br tag is used to insert a "line break" which means the text/image following the tag will be moved to the next line.
This is an invisible separator for the ones that see the web version, but it is seen and read by the browser and applied as an occupied space in page.

In between this 2 lines the break occurs.


In between this 2 lines the break occurs.

The Code: <br/>

The hr tag is used to insert horizontal line that can divide the information into sections.
This is a visible separator for both the one seeing the web version and the ones that seeing the code behind it.

The Code: <hr/>
