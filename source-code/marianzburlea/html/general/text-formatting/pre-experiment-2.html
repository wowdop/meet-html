Making use of the pre html tag

The HTML for the header and this paragraph will all be displayed below in a pre tag. 

  <h1>Making use of the <i>pre</i> html tag</h1>
  <p>This will all be displayed below in a <b>pre tag</b></p>

The code looks like this:

<pre>
  <h1>Making use of the <i>pre</i> html tag</h1>
  <p>This will all be displayed below in a <b>pre tag</b></p>
</pre>
