The s tag specifies text that is no longer correct, accurate or relevant. The text will be displayed with a line through it.

Most browsers will display the <s> element with the following default values: Values

Code: <s style="text-decoration: line-through">Values</s>

Just another text example.

Code: <p>Just another <s>text</s> example.</p>

The del tag defines text that has been deleted from a document. Browsers will usually strike a line through deleted text.

It is usually used when you want to emphasize a deleted part of the text and the new insert for the deleted part.

My favorite place to spend my time is in the park bed.

The code: <p>My favorite place to spend my time is in <del>the park</del> <ins>bed.</ins></p>

The strong is used to define text with strong importance. The content inside is typically displayed in bold. While we use the b tag to specify bold text without any extra importance!

Example: This is a Strong tag vs. a Bold tag.

The code: This is a <strong>Strong</strong> tag vs. a <b>Bold</b> tag.

The b tag specifies bold text without any extra importance.

Note: According to the HTML5 specification, the <b> tag should be used as a LAST resort when no other tag is more appropriate.

Code: <b>Note</b>: According to the HTML5 specification, the <b> tag should be used as a LAST resort when no other tag is more appropriate.

Tip: You can also use the following CSS to set bold text: "font-weight: bold;".

The em tag is used to define emphasized text. The content inside is typically displayed in italic.

A screen reader will pronounce the words in <em> with an emphasis, using verbal stress.

Most browsers will display the <em> element with the following default values:{font-style: italic;}

The i The tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.

Use the <i> element only when there is not a more appropriate semantic element, such as: <em> (emphasized text) <strong> (important text) <mark> (marked/highlighted text) <cite> (the title of a work) <dfn> (a definition term)

A kitty hid inside the backpack.

Code: A <i>kitty hid inside the backpack.</i>

The u tag usually results in the text being underlined (unless you style it otherwise). For this reason, you should avoid using this element in cases where it may be confused for a hyperlink.

Usualy, the <u> is used with { text-decoration: underline; }
