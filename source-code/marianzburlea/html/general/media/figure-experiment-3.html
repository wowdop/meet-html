Make use of figure to display a peom from someone

Let's create a figure tag and inside to make use of figcaption.

Then let's add some style so whoever looks at it will read it super easy.

I have been looking online for a nice poem about nymphs and this is what came up.

Code example:

  <style>
    figure {
      border: 5px ridge #847171;
      flex-flow: column;
      padding: 0.5rem;
      width: 250px;
    }

    figcaption {
      background-color: rgb(6, 40, 109);
      color: #fff;
      font: italic smaller sans-serif;
      padding: 3px;
      padding: 0.5rem;
    }
  </style>
  <figure>
    <p style="white-space: pre">
      Hold fast to dreams <br />
      For if dreams die <br />
      Life is a broken-winged bird <br />
      That cannot fly. <br />
      Hold fast to dreams <br />
      For when dreams go <br />
      Life is a barren field <br />
      Frozen with snow.
    </p>
    <figcaption><cite>Dreams</cite>, by Langston Hughes</figcaption>
  </figure>
