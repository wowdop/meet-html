Make use of figure to display an image with some text

Let's create a figure tag and inside to make use of figcaption and img tags.

Then let's add some style so whoever looks at it will find it appealing.

Code example:

  <style>
    figure {
      border: thin #949494 solid;
      padding: 1rem;
      width: 220px;
    }
  </style>

  <figure>
    <img
      width="220"
      height="150"
      src="https://picsum.photos/200/300/?blur"
      alt="This will always be a random image with blur"
    />
    <figcaption>This is a lovely blurred picture</figcaption>
  </figure>
