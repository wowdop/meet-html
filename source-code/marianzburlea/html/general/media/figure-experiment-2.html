Make use of figure to display a quote from someone

Let's create a figure tag and inside to make use of figcaption and blockquote.

Then let's add some style so whoever looks at it will read it super easy.

Today I was doing some research and I notice this interesting quote that gave me something to think about.

Code example:

  <style>
    figure {
      border: 5px ridge #847171;
      padding: 0.5rem;
    }
  </style>

  <figure>
    <figcaption><cite>Edsger Dijkstra:</cite></figcaption>
    <blockquote>
      If debugging is the process of removing software bugs, then programming must
      be the process of putting them in.
    </blockquote>
  </figure>
