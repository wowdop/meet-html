This is an video file embeded using the embed tag

I made use of a simple embed tag and I changed the value for attribute type to video/webm

Let's try something else

Let's embed a different video file and style it with some border to make it more visible for an user.

This video is representing how multiple people can work in the same time on Figma.

This allows people to become more efficient and the design can be finished earlier.

It is also an advantage as this allows all the team members to access the design any time and see real time updates as long as they are connected to the internet.

Code example:

  <style>
    div {
      border: 0.5rem dotted crimson;
      padding: 0.5rem;
    }
  </style>

  <embed
    src="../../../../public/video/remove-bg.mp4"
    width="100%"
    height="300"
    type="video/webm"
  />

  <div>
    <h3>
      This video is representing how multiple people can work in the same time on
      Figma.
    </h3>

    <p>
      This allows people to become more efficient and the design can be finished
      earlier.
    </p>

    <p>
      It is also an advantage as this allows all the team members to access the
      design any time and see real time updates as long as they are connected to
      the internet.
    </p>
    <embed
      src="../../../../public/video/realtime-edit.mp4"
      type="video/webm"
      width="500"
      height="200"
    />
  </div>
