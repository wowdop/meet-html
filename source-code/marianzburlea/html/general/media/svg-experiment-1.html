You can create a simple rectangle

Code example:
    <svg width="200" height="100">
      <rect
        width="200"
        height="100"
        style="fill: rgb(92, 150, 106); stroke-width: 8; stroke: rgb(164, 75, 75)"
      />
    </svg>

Or even add some text in it if you want to make it look like a logo

Note: We made use of text tag as it is the only one that is accepted in the svg tag. You can find more details in the README.md file

Code example:
    <style>
      .logo {
        height: 60px;
        width: 100;
        background-color: #5377ac;
        border: 1px solid grey;
        border-radius: 0.5rem;
        box-shadow: 2px 2px rgb(62, 62, 161);
      }

      text {
        text-shadow: 1px 1px whitesmoke;
      }
    </style>

    <svg class="logo">
      <text fill="#c23d66" font-size="25" font-family="Verdana" x="22" y="38">
        SVG
      </text>
    </svg>
