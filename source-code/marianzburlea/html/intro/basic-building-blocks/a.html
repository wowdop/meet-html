Welcome to link in HTML

A link in HTML is used to bridge together two resources: a page with an image, a page with another page and so on.

Linking from a page of a web site to another page of the same web site is called internal linking.

If a link from an web site points to a resource outside the web site of origin, than we can talk about external links.

If you want to visit Google to do a search, visit https://google.com as this is an external link. Links that have a target attribute with the value of _blank will open on click the web page it points to in a new window.
