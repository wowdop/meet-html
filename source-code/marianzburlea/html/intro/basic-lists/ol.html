Use ol tag when you have lists that should be ordered.

You can specify the kinf of marker to use in the list either from CSS or by adding the attribute type to the ol tag

type can have these values:

type="a"

Lower letters order

type="A"

Uppercase letters
Groceries list Chapters to read
  1. Bananas
  2. Pineapple
  3. Strawberries
  4. Pizza
  1. Entering the dungeon
  2. The messy hell
  3. Painful pity
  4. Reborn

type="i"

Lowercase Roman numeral

type="I"

Uppercase Roman numeral
Places to visit Countries available
  1. Hawaii
  2. Filipines
  3. Indonesia
  4. Thailand
  5. Japan
  6. Bahamas
  1. Spain
  2. Ireland
  3. Italy
  4. Venezuela

type="1"

Numbers
Daily tasks
  1. Clean dishes
  2. Throw the garbage
  3. Buy groceries
  4. Cook

Code:

      <ol type="a">
        <li>Bananas</li>
        <li>Pineapple</li>
        <li>Strawberries</li>
        <li>Pizza</li>
      <ol>
