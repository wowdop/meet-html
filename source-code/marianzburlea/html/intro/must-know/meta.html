Metadata

Should be placed in the head tag

Metadata is data about the HTML document. Metadata is not displayed.

Metadata typically define the document title, character set, styles, scripts, and other meta information.

Search engines love a good reason to visit your website

      <meta 
        name="description" 
        content="This is a message that fits the reason to click and visit"
      >

Developing for smart phones and many browsers require some extra

      <meta charset="UTF-8">
      <meta http-equiv="X-UA-Compatible" content="IE=edge">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
