The html tag represents the root of an HTML document and it is the container for all other HTML elements (except for the <!DOCTYPE> tag).

Note: You should always include the lang attribute inside the html tag, to declare the language of the Web page. This is meant to assist search engines and browsers.

The head element is a container for metadata (data about data) and is placed between the html tag and the body tag.

The body tag must be the second element after the head tag.

It's always placed between the html tags.

This tag is required for every HTML document and should only use once in the whole HTML document. 

      <!DOCTYPE html>
      <html lang="en">
      <head></head>
      <body></body>
      </html>
