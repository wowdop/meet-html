Display images using HTML tag

img tag needs to have src attribute to specify the source of the image and alt for a short description that can be visible if the image is not loading.

Note:This is also for the voice over assistant in case of disabilities.

Code:

      <img src="https://picsum.photos/200/300" alt="This will always be a random picture">
